# 🏗️ BIM Project Manager 2.0

## Descripción

BIM Project Manager 2.0 es una aplicación web desarrollada como proyecto académico para la gestión básica de proyectos BIM.

Permite registrar proyectos, asignar responsables, seleccionar el estado del proyecto y visualizar la información de forma organizada mediante una interfaz moderna.

---

## Funcionalidades

- ✅ Agregar proyectos.
- 👤 Registrar responsable.
- 📅 Fecha automática de creación.
- 📌 Estado del proyecto.
- 📊 Contador de proyectos.
- 🗑️ Eliminar proyectos.
- 💻 Interfaz moderna y responsive.

## Cómo ejecutar el proyecto

1. Descargar el proyecto.
2. Abrir el archivo **index.html** en cualquier navegador.
3. Registrar un proyecto.
4. Consultar la lista de proyectos.
5. Eliminar proyectos cuando sea necesario.
