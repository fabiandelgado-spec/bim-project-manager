# BIM Project Manager

## Descripción

Aplicación web básica desarrollada para la gestión de proyectos BIM como práctica académica.

## Funcionalidades

- Agregar proyectos.
- Visualizar proyectos registrados.
- Eliminar proyectos.
- Interfaz sencilla y fácil de usar.

## Tecnologías utilizadas

- HTML5
- CSS3
- JavaScript

## Cómo ejecutar el proyecto

1. Descargar el proyecto.
2. Abrir el archivo **index.html** en cualquier navegador.
3. Escribir el nombre de un proyecto.
4. Presionar el botón **Agregar Proyecto**.
5. El proyecto aparecerá en la lista.
6. Puede eliminarse usando el botón **Eliminar**.
